# Social Studio Backlog

This document tracks future features and improvements for the Social Studio App.

## High Priority 🚀
- [ ] **Instagram Reels Native Integration**: Direct API connection for cross-posting Reels.
- [ ] **TikTok Native Integration**: Direct API connection for vertical video distribution.
- [ ] **AI Vibe-Writer**: One-click caption rewriting (Professional for LinkedIn, Chaotic for TikTok, etc.).
- [ ] **Media Gallery**: A centralized view to manage local and uploaded video files.

## Medium Priority 📈
- [ ] **Analytics Dashboard**: Fetch and display performance metrics (Views, Reach, Engagement) from YouTube/Instagram.
- [ ] **Scheduling & Calendar**: Plan out posts across multiple platforms with a visual calendar.
- [ ] **Music Trending Engine**: Suggest trending audio for native app selection or AI beat-sync suggestions.

## Low Priority 🛠️
- [ ] **Multi-Account Support**: Manage multiple YouTube channels or Instagram accounts per user.
- [ ] **Video Preview**: Integrated video player to review content before posting.
- [ ] **Dark/Light Mode Toggle**: Professional theme switching.

## Completed ✅
- [x] **YouTube Direct Upload**: OAuth2 integration for Shorts/Videos.
- [x] **Header Navigation**: "Create Post" active scrolling.
- [x] **Initial Project Structure**: Next.js + Tailwind (v4) setup.
